# 월급계산기 1.1.0

- 앱 안에서 새 버전을 다운로드하고 설치할 수 있는 자동 업데이트 기능을 추가했습니다.
- 다운로드한 파일은 전용 암호 서명을 검증한 뒤에만 설치합니다.
- 설치가 끝나면 기존 앱을 새 버전으로 교체하고 자동으로 다시 실행합니다.
- 계정, 급여 조건, 공제 설정, 월별 근태 데이터는 업데이트 후에도 그대로 유지됩니다.

기존 1.0.x 버전에는 자동 교체 기능이 없으므로 이번 1.1.0 DMG만 한 번 직접 설치해야 합니다. 이후 버전부터는 앱 안에서 업데이트할 수 있습니다.

Apple Developer 서명·공증을 하지 않은 배포본이므로 최초 실행 시 macOS 보안 경고가 표시될 수 있습니다.
