# 월급계산기 1.2.0

- 입력한 연봉과 비과세 항목을 기준으로 4대보험 예상 기준금액을 사람별로 다시 계산합니다.
- 2026년 공식 근로소득 간이세액표 전체 구간을 반영해 연봉이 바뀌면 소득세도 자동으로 변경됩니다.
- 공제대상가족 1~11명, 8~20세 자녀 수, 원천징수 80%·100%·120% 설정을 적용합니다.
- 회사의 실제 신고 기준금액이 있으면 `공제 설정 → 신고값 직접 입력`으로 명세서와 맞출 수 있습니다.
- 계정, 급여 조건, 공제 설정, 월별 근태 데이터는 업데이트 후에도 그대로 유지됩니다.

1.1.0 이상을 사용 중이라면 앱의 `업데이트 확인` 버튼으로 설치할 수 있습니다.

Apple Developer 서명·공증을 하지 않은 배포본이므로 최초 실행 시 macOS 보안 경고가 표시될 수 있습니다.
